Project Experience

Movie Recommender System


Tech Stack: Python, Ollama, FAISS, Streamlit, NGROK

Techniques: Ollama embeddings, FAISS vector stores, sub-second similarity search, UI app, Self-hosting

Description: Built a full-stack movie recommender that preprocesses and indexes ~23k titles across five catalogs (Netflix, Amazon, Hulu, Disney, combined) into FAISS vector stores for sub-second similarity search. Designed a reusable backend pipeline generating L2-normalized embeddings via a local Ollama model, optimizing workflow to cut recommendation prep from minutes to seconds. Developed a Streamlit frontend with integrated similarity scoring, deployed locally and shared securely via ngrok, with automation, deployment, and debugging practices documented for reproducibility.

Quora Question Pairs


Tech Stack: NLTK, Spacy, Matplotlib, Linear Regression, SVM, Neural Networks, XGBoost

Techniques: Data Augmentation, Data Visualization, Data Analysis, Predictive Modelling

Description: The project revolves around addressing the challenge of information overload on the knowledge-sharing platform Quora. With the exponential growth of data, the goal is to identify and manage duplicate questions on Quora, enhancing the user experience by preventing the degradation of answer quality. The machine learning component focuses on pairing new questions with existing ones, employing features like basic statistics, text preprocessing, and advanced NLP techniques. The dataset, sourced from Kaggle, consists of labeled question pairs for training and evaluation. Baseline models, such as DummyClassifier, establish a benchmark for performance evaluation. Further analysis involves exploring data distribution, feature engineering, and implementing machine learning models like Logistic Regression, Support Vector Machine, XGBoost, and a Deep Neural Network using TensorFlow. The best-performing model achieves an accuracy of 79.29%, demonstrating its effectiveness in identifying duplicate question pairs on Quora.

Blind Vision


Tech Stack: Python, Computer Vision, HTML, CSS, Javascript

Techniques: API Integration, Virtual Assistance, Object Detection, SMS Communication, Speech-To-Text, Text-To-Speech

Description: The Blind Vision project is a groundbreaking initiative aimed at enhancing the mobility of visually impaired individuals by serving as their virtual eyes. Utilizing a Raspberry Pi 3 with a Pi Cam module, complete with built-in wifi and Bluetooth capabilities, the system is seamlessly integrated into a walking stick for user convenience. Employing the YOLO Object detection system locally minimizes response time, allowing real-time detection of objects and distance approximation, communicated to the user via a connected headphone module. Face Recognition


Tech Stack: Python, OpenCV

Techniques: Image Processing, Face Recognition, Real-Time Face Recognition

Description: The Face Recognition project is a sophisticated computer vision initiative designed to identify and outline faces within various contexts, including images, videos, or real-time scenarios. Powered by Python and utilizing the OpenCV computer vision library, the system is adept at employing the Cascade Classifier to robustly detect multiple faces within the captured media. Adding a layer of precision to the facial detection process, the project incorporates the LBPH (Local Binary Pattern Histogram) Face Recognizer. This specialized tool accurately identifies the regions of faces, providing detailed coordinates that specify the rectangular region of interest around each detected face. OpenCV's foundational functions are utilized to draw clear and distinct outlines around the detected faces, using the precise coordinates obtained from the LBPH Face Recognizer. In essence, this project showcases the seamless integration of Python, OpenCV, and advanced facial recognition techniques to deliver a robust and visually compelling solution.

Flower Classification (HackerEarth Challenge)


Tech Stack: Python, Keras, Pandas

Techniques: Image Segregation, Image Preprocessing, Neural Network modeling, Model Training and Testing

Description: Embarking on the Flower Classification challenge within HackerEarth's Deep Learning series, our approach involved meticulous steps to ensure accurate flower image classification. We began by organizing the dataset, segregating images into specific class folders. Converting these images into numpy arrays, we then implemented image data generators to augment the model's exposure to new variations at each training epoch. For the core of our model, we opted for the VGG16 neural network, known for its effectiveness. Our choice of the Adam optimizer and categorical cross-entropy loss function aimed at optimizing the model's performance. Through rigorous training and testing phases on the provided datasets, we aimed to create a robust solution for precise flower classification.

Clustering Geographical Data


Tech Stack: Python, Scikit-learn

Description: The "Clustering Geographical Data" project is a comprehensive exploration of taxi rank locations, employing Python and various libraries for data analysis and clustering.
